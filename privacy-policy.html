Политика конфиденциальности

Дата вступления в силу: 7 февраля 2026 г.

Настоящая Политика конфиденциальности описывает, каким образом приложение VisitManager (далее — «Приложение») обрабатывает информацию пользователей.

1. Сбор данных

Приложение не собирает, не передаёт и не хранит персональные данные на внешних серверах. Все данные, вводимые пользователем (информация о пациентах, визитах, осмотрах), хранятся исключительно локально на устройстве пользователя.

2. Хранение данных

Данные сохраняются в локальном хранилище устройства (AsyncStorage) и доступны только через Приложение. Приложение не использует облачные сервисы, серверы или базы данных для хранения пользовательских данных.

3. Передача данных третьим лицам

Приложение не передаёт никакие данные третьим лицам, рекламным сетям, аналитическим сервисам или иным внешним системам. Приложение работает полностью автономно (офлайн).

4. Разрешения устройства

Приложение может запрашивать следующие разрешения:

  • Камера — для фотографирования в рамках осмотра пациента.
  • Галерея (фотоплёнка) — для прикрепления существующих фотографий к записям осмотра.

Фотографии хранятся локально на устройстве и не передаются на внешние серверы.

5. Безопасность

Поскольку все данные хранятся локально на устройстве, их безопасность обеспечивается средствами защиты операционной системы (пароль, Face ID, Touch ID). Мы рекомендуем пользователям использовать блокировку экрана для дополнительной защиты.

6. Данные детей

Приложение не предназначено для использования лицами младше 17 лет и не собирает данные о детях.

7. Удаление данных

Пользователь может удалить все данные Приложения в любой момент, удалив приложение с устройства или очистив данные приложения через настройки операционной системы.

8. Изменения в политике

Мы оставляем за собой право обновлять настоящую Политику конфиденциальности. Актуальная версия всегда доступна по данному адресу. Продолжение использования Приложения после внесения изменений означает согласие с обновлённой Политикой.
